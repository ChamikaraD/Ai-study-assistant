# AI Study Assistant

A multimodal AI-powered mobile application built using Flutter and FastAPI.

## Features
- Upload Notes (PDF, TXT)
- Record Lectures (Speech-to-Text)
- AI Summarization
- Context-based Q&A
- Study History

## Tech Stack
- Flutter
- FastAPI
- OpenAI / Whisper
- Firebase / Supabase

## Architecture
Flutter → FastAPI → AI Models
